# dotfiles

Personal configuration files for my Linux setup.

Website: <https://tsk-io.github.io/dotfiles/>

## Contents

| Directory   | Description                          |
| ----------- | ------------------------------------ |
| `dunst`     | Notification daemon configuration    |
| `fcitx5`    | Input method configuration           |
| `fish`      | Fish shell configuration             |
| `git`       | Git configuration                    |
| `gnupg`     | GnuPG configuration                  |
| `icewm`     | IceWM window manager configuration   |
| `kitty`     | Kitty terminal configuration         |
| `starship`  | Starship prompt configuration        |
| `vim`       | Vim configuration                    |
| `scripts`   | Helper scripts                       |
| `docs`      | Documentation (published as website) |
