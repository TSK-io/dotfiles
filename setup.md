## Debian 最小化安装后恢复网络及安装桌面环境教程
**适用场景：**
在安装 Debian 12（或更新版本）时，如果在“软件选择”界面没有勾选“Debian desktop environment”和“standard system utilities”，系统启动后将进入纯命令行界面。此时图形化网络管理工具（NetworkManager）未安装，且新版 Debian 移除了传统的 dhclient 工具，导致网络默认断开。
**前置准备：**
请确保你的电脑已经**插上网线**（连接到路由器），并且使用 root 账户登录系统（或在所有命令前加上 sudo）。
### 第一步：确认网卡接口名称
首先，我们需要找到有线网卡的系统代号。
 1. 输入以下命令查看网络接口：
   ```bash
   ip a
   
   ```
 2. 在输出结果中找到你的有线网卡。它通常以 en 或 eth 开头（例如：enp3s0、eno1、eth0）。请记下这个名称，后面的步骤会用到。
### 第二步：启用网卡并获取 IP 地址
由于缺少旧版的 dhclient，我们将使用 Debian 系统中自带的 dhcpcd 客户端来自动获取 IP。
 1. 启动你的有线网卡（请将 enp3s0 替换为你实际的网卡名称）：
   ```bash
   ip link set enp3s0 up
   
   ```
 2. 使用 dhcpcd 命令向路由器请求分配 IP 地址：
   ```bash
   dhcpcd enp3s0
   
   ```
   > **注意：** 命令执行后，系统会在后台向路由器请求 IP。如果你看到屏幕上返回了包含 IP 地址（如 192.168.x.x）的信息，说明获取成功。
   > 
### 第三步：测试网络连通性
在进行软件安装前，确认一下网络是否真的畅通。
 1. 尝试 ping 外网服务器（例如 Debian 官网或百度）：
   ```bash
   ping -c 4 debian.org
   
   ```
 2. 如果屏幕输出类似 64 bytes from ... time=... ms 的信息，说明网络已成功恢复！
### 第四步：安装图形化网络管理器
为了确保进入桌面环境后右下角有网络图标，且能方便地连接 Wi-Fi，我们需要先安装图形化的网络管理工具。
 1. 更新系统的软件源列表：
   ```bash
   apt update
   
   ```
 2. 安装 NetworkManager：
   ```bash
   apt install network-manager
   
   ```
### 第五步：补装桌面环境
网络和基础工具准备就绪后，就可以调出安装系统时的软件选择界面，把漏掉的桌面环境装回来了。
 1. 运行 Tasksel 界面：
   ```bash
   tasksel
   
   ```
 2. 在出现的蓝色图形界面中，使用**上下方向键**移动光标，按**空格键**勾选以下选项（选项前面出现星号 * 即为选中）：
   * Debian desktop environment（必选，基础桌面支持）
   * 你喜欢的任意桌面环境（如 GNOME、KDE Plasma、Xfce 等，选一个即可）
   * standard system utilities（建议勾选，包含一些常用系统工具）
 3. 按 **Tab 键** 切换到 <Ok> 或 <确定> 按钮，然后按 **回车键** 确认。
 4. 耐心等待系统自动下载并安装所需的大量软件包。
### 第六步：重启系统
所有软件安装完成后，重启电脑：
```bash
reboot

```
重启后，系统将直接进入你熟悉的图形化桌面登录界面，并且网络也会由系统自动接管。
