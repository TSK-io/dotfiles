📦 核心组件

🐚 Fish Shell

现代化的 shell，支持 Vi 键绑定、fzf 集成、剪贴板感知编辑

📺 Tmux

终端复用器，支持 Vi 复制模式、快速快捷键、弹出式 fzf 选择器

🪟 i3 窗口管理器

极简键盘驱动的窗口管理，支持工作区、快捷键、带 Rofi 启动器

📝 Git 配置

GitHub CLI 集成、SSH 提交签名、优化的路径显示和别名

🎨 Alacritty

GPU 加速终端，性能优秀，配置灵活，与 Tmux 完美搭配

⚙️ 自动化脚本

完整的 Debian 系统设置脚本，一键配置桌面和常用工具

🚀 快速开始

1

克隆仓库

从 GitHub 获取完整的配置文件

git clone https://github.com/tsk-io/dotfiles.git
2

选择配置

根据需要符号链接或复制相关目录

ln -s dotfiles/fish ~/.config/
3

运行设置脚本

可选：自动安装所需的工具和依赖

./scripts/.local/bin/extend_setup.sh
4

自定义调整

根据个人偏好修改配置，开始工作

vim ~/.config/fish/config.fish
